# grid-performance App

This project can be used to test performance of adding and removing columns to/from a Grid.

## Running the application

Open the project in an IDE. You can download the [IntelliJ community edition](https://www.jetbrains.com/idea/download) if you do not have a suitable IDE already.
Once opened in the IDE, locate the `Application` class and run the main method using "Debug".

## Testing the performance

You will find an input field 'column count', a button 'Generate', and a Grid.
Input 500 as column count, press Generate.
Input 501 as column count, press Generate - and wait

If it is not boring enough try higher numbers
